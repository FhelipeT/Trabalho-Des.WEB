# salao da netinha
Salão da Netinha

Projeto desenvolvido para a disciplina Desenvolvimento Web em HTML, CSS e Javascript, com foco na criação de um website responsivo e funcional.

 Descrição do Projeto

 Uso de HTML5, CSS, Javascript e Bootstrap
 Interface responsiva
 Navegação Fluida
 Interatividade com JS

 Tecnologias Utilizadas

HTML
CSS
JavaScript
Bootstrap
Filtro de serviços
Busca por texto
Sistema de agendamento
Verificação de login
JSON
Google Fonts


 Funcionalidades
Catálogo de Serviços
Lista dinâmica carregada via servicos.json
Cards com preço, imagem, categoria e duração
Botão de “Agendar”

 Busca e Filtros

Busca instantânea por nome do serviço
Categorias: cabelos, unhas, estética, maquiagem
Botões com filtro automático

 Sistema de Agendamento

Formulário com dados validados
Salva agendamentos localmente com localStorage
Exibição dos agendamentos na tela
Botão para limpar todos os agendamentos

 Sistema simples de Login

Usuário precisa estar logado para acessar o site
Senão é redirecionado automaticamente para login.html

 Navegação

Botões navegáveis e rolagem suave
Redirecionamento automático para áreas internas
